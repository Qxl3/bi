# 项目简介

## 项目介绍

基于React+ pring Boot + MO + AIGC 的智能数据分析平台。
区别于传统 BI，用户只需要导入原始数据集、并输入分析诉求，就能自动生成可视化图表及分析结论，实现数据分
析的隆本增效。
